# StepClean Taubaté

Landing page moderna desenvolvida para a **StepClean Taubaté**, uma lavanderia premium de tênis especializada em limpeza, remoção de manchas, tratamento de amarelados, impermeabilização, hidratação e repintura de sneakers.

## Sobre o projeto

O objetivo deste projeto é apresentar os serviços da StepClean de forma visual, moderna e responsiva, com foco em conversão pelo WhatsApp.

A página foi construída com uma identidade visual escura, elementos em azul e branco, animações suaves, cards de serviços, FAQ interativo e uma experiência visual de antes/depois no hero da página.

## Funcionalidades

* Layout moderno e responsivo
* Botões com direcionamento para WhatsApp
* Menu com rolagem suave
* FAQ interativo
* Animações ao rolar a página
* Efeito visual de antes/depois no tênis
* Seção de serviços
* Seção de processo de atendimento
* Rodapé com links para Instagram e WhatsApp

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages
